== Description ==

Need a ranking? We make it easy.
Use the TheRanking.com pluggin. Choose any of the rankings that we offer by category, or if you prefer, create your own ranking and then ask your users that vote.
In TheRanking.com we certify the voting process, so we track against Facebook, Twitter or username/password.
Easy, useful and fun.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
